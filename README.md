# EquiTrade · Advanced Trading Dashboard

A modern stock trading platform built with **Spring Boot** backend and **HTML/CSS/JavaScript** frontend. Buy, manage, and track your stock portfolio with an intuitive interface.

## Features

- 📊 **Live Stock Market**: Browse 20 pre-loaded stocks with real-time pricing in Indian Rupees (₹)
- 🛒 **Buy Stocks**: Execute instant stock purchases with quantity selection
- 💼 **Portfolio Tracking**: View your holdings with real-time updates
- 🔐 **User Authentication**: Register and login with JWT-based security
- 🎛️ **Admin Control**: Manage stock listings and remove outdated entries
- 🔍 **Stock Search**: Filter stocks by symbol or company name
- 📱 **Responsive Design**: Clean, modern UI optimized for all devices

## Tech Stack

### Backend
- **Java 11+**
- **Spring Boot** (REST API framework)
- **Spring Security** (JWT authentication)
- **Spring Data JPA** (database access)
- **H2 Database** (in-memory)
- **Maven** (build tool)

### Frontend
- **HTML5** (semantic structure)
- **CSS3** (modern styling with gradients and animations)
- **JavaScript ES6+** (dynamic interactions)
- **Fetch API** (HTTP requests)
